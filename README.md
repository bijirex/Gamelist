# Kids Team Game Board

A fun interactive game selector for kids team games.
Built with plain HTML, CSS, and JavaScript — no dependencies.

## How to use
1. Open `index.html` in any browser
2. Select number of kids
3. Browse games and click to see instructions

## Games included
- Relay race
- Tug of war
- Treasure hunt
- Balloon pop relay
- Human knot
- Freeze tag
- Hula hoop pass
- Capture the flag
